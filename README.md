# Ximing Tao - Technical Storytelling Portfolio

Static Astro site for `ttm43/ximingtao.github.io`.

## Structure

- Home: positioning, proof metrics, selected systems
- Case Study 01: Order Processing System
- Case Study 02: Controlled Knowledge System
- Case Study 03: Document Intelligence
- Resume: web-native resume with print/PDF button

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

GitHub Actions deploys `dist/` to GitHub Pages on pushes to `main`.
